Lazy Loading Implementation

Lazy loading in Angular Router is implemented as follows:

// In app-routing.module.ts
const routes: Routes = [
  // Other routes...
  {{ '{' }} 
    path: 'user', 
    loadChildren: () => import('./modules/user/user.module')
      .then(m => m.UserModule)
  {{ '}' }}
];

Check your browser's network tab to see that the module chunk was loaded only when visiting this route.
